> [!IMPORTANT]
>
> - Teams AI library is now renamed to Teams SDK.
> - Teams AI library v1 is deprecated. We recommend that you upgrade your agents to use the updated [Teams SDK](../teams-sdk-overview.md).
>
> Teams SDK is generally available for TypeScript, C#, and Python. It provides a simplified SDK, support for Model Context Protocol (MCP), Agent-to-Agent communication (A2A), and streamlined tools to enable developers to build intelligent agents for Teams.
