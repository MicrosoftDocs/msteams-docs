# Enable slash commands for agents

Slash commands are a lightweight way to invoke an agent's or bot's capabilities using a predictable, text-first syntax that starts with '/' (slash). Users can type a short command such as `/summarize`, `/draft`, or `/create-task` directly in the message compose box to trigger an action immediately. Slash commands act like power shortcuts that combine the speed of commands with the flexibility of natural language.
